# Quick Start Guide

Get SupportHub running in 3 simple steps!

## 🚀 Step 1: Install Node.js

**Download and install Node.js from [https://nodejs.org/](https://nodejs.org/)**
- Choose the **LTS** (Long Term Support) version
- Run the installer and follow the setup wizard
- **Restart your computer** after installation

## 📦 Step 2: Install Dependencies

**Option A: Using the provided script (Recommended)**
- Double-click `install.bat` (Windows) or `install.ps1` (PowerShell)
- Wait for installation to complete

**Option B: Manual installation**
- Open Command Prompt or PowerShell in this folder
- Run: `npm install`

## 🎯 Step 3: Start the Application

**Option A: Using the provided script (Recommended)**
- Double-click `start.bat` (Windows) or `start.ps1` (PowerShell)

**Option B: Manual start**
- Open Command Prompt or PowerShell in this folder
- Run: `npm run dev`

## 🌐 Access Your App

- Open your web browser
- Navigate to: **http://localhost:3000**
- You should see the SupportHub landing page!

## 🆘 Need Help?

If you encounter any issues:

1. **Check the troubleshooting guide**: `TROUBLESHOOTING.md`
2. **Verify Node.js installation**: Run `node --version` in terminal
3. **Check for error messages**: Look at the terminal output
4. **Common issues**: 
   - Node.js not installed → Install from nodejs.org
   - Port 3000 in use → Close other applications or change port
   - Dependencies missing → Run `npm install`
